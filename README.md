# alhamdan-digital

Al Hamdan Digital marketing website built with Next.js, TypeScript, Tailwind CSS, and next-intl (Arabic/English).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for Arabic (default) or [http://localhost:3000/en](http://localhost:3000/en) for English.

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run start` — run production server
- `npm run lint` — run ESLint
